# LUXE Store - Modern E-commerce Platform

A modern, responsive e-commerce website built with vanilla HTML, CSS, and JavaScript.

## Features

- **Modern UI/UX** - Clean, attractive design with smooth animations
- **Fully Responsive** - Works on all devices (mobile, tablet, desktop)
- **Product Catalog** - Browse products with filtering and sorting
- **Shopping Cart** - Add/remove items with persistent storage
- **Checkout Flow** - Multi-step checkout process
- **Contact Form** - Get in touch functionality
- **User Authentication** - Login/Register interface

## Pages

- `index.html` - Homepage with hero slider, categories, featured products
- `shop.html` - Product catalog with filters and sorting
- `cart.html` - Shopping cart management
- `checkout.html` - Multi-step checkout process
- `about.html` - About us page
- `contact.html` - Contact form and information
- `login.html` - User authentication

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Font Awesome 6.4.0
- Google Fonts (Poppins, Playfair Display)
- LocalStorage for cart persistence

## Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/luxe-store.git
   cd luxe-store
   ```

2. Open `index.html` in your browser, or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

3. Visit `http://localhost:8000`

## Deploy to Vercel

### Option 1: Using Vercel CLI

1. Install Vercel CLI:
   ```bash
   npm install -g vercel
   ```

2. Deploy:
   ```bash
   vercel
   ```

### Option 2: Using Vercel Dashboard

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Click "Deploy"

The site will be automatically deployed and you'll get a live URL.

## Deploy to GitHub Pages

1. Go to your repository Settings
2. Navigate to Pages section
3. Select "main" branch as source
4. Save and wait for deployment
